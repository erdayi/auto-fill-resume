# 简历自动填写助手 - Resume Auto Filler

Chrome / Edge 浏览器插件，一键将个人简历信息智能填入求职网站表单。

## 功能特性

- **智能识别** — 多信号加权评分，自动匹配表单字段（name/id/label/placeholder 等 15+ 信号源）
- **多条目填写** — 教育经历、工作/实习、项目、竞赛/荣誉、证书/资质均支持多条添加
- **框架兼容** — 支持 React、Vue 等前端框架的表单，自动触发合成事件
- **自定义组件** — 兼容 Ant Design、Element UI、Arco Design 等 UI 库的日期选择器和下拉框
- **快捷键** — `Alt+Shift+F` 一键填写当前页面
- **数据管理** — 自动保存、JSON 导入/导出，数据存储在浏览器本地
- **双浏览器** — 同时兼容 Chrome 和 Edge

## 安装使用

### 1. 下载代码

```bash
git clone https://github.com/erdayi/auto-fill-resume.git
```

### 2. 加载插件

**Chrome:**
1. 打开 `chrome://extensions/`
2. 右上角打开 **开发者模式**
3. 点击 **加载已解压的扩展程序**
4. 选择项目文件夹 `auto-fill-resume`

**Edge:**
1. 打开 `edge://extensions/`
2. 左下角打开 **开发人员模式**
3. 点击 **加载解压缩的扩展**
4. 选择项目文件夹 `auto-fill-resume`

### 3. 填写个人信息

- 点击浏览器右上角插件图标，打开弹窗
- 在 5 个标签页中填写你的信息：基本信息 | 教育经历 | 工作/实习 | 项目/竞赛 | 其他
- 数据会自动保存

### 4. 一键填写

- 打开任意求职网站的表单页面
- 点击插件弹窗中的 **一键填写** 按钮
- 或使用快捷键 `Alt+Shift+F`

## 技术原理

1. **字段识别** — 提取表单元素的 name、id、label、placeholder 等 15+ 种信号，与 50+ 个字段匹配器进行加权评分，阈值 ≥ 8 分判定匹配
2. **值设置** — 通过原生 prototype setter + 合成事件确保 React/Vue 等框架能响应值变更
3. **多条目** — 检测页面分区（教育/工作/项目等），按容器分组填写，自动点击"添加"按钮创建新条目
4. **自定义组件** — 识别 Ant Design、Element UI 等 UI 库的日期选择器和下拉框，模拟交互操作
